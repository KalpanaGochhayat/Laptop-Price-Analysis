# Laptop Price Analysis <br>

**Author:** **_Kalpana_**  <br>

A data analysis project exploring laptop specifications and their impact on pricing. The goal is to** analyze features**, **visualize trends**, and **build predictive models** to estimate laptop prices.
<br><br>
## Tools & Software

- **Python 3.x**  
- **Jupyter Notebook**  
- **Pandas** (data manipulation)  
- **NumPy** (numerical computation)  
- **Matplotlib** & **Seaborn** (visualization)  
- **Scikit-learn** (machine learning)
